sobre

Sempre fui curioso quando o assunto se tratava de tecnologia. Minha vontade por fazer parte de TI e desenvolvimento surgiu quando eu me questionei pela primeira vez: "O que tem por trás de um game? Como é todo esse processo?". Após descobrir que os códigos são um dos pilares de uma arte como os games, eu disse a mim mesmo que iria trabalhar na área de dev um dia.

Em 2017 tive meu primeiro contato com programação, quando fiz um curso de C#, mas não continuei. Só em 2020 voltei a área, quando consegui minha vaga em Engenharia de Software na PUC-PR, onde tive breve contato com Python, Java, mas após 6 meses tive que trancar a faculdade. Nesse ano de 2022 voltei a estudar. Após bastante pesquisa, decidi iniciar pelo Front-End, mas minha meta é me tornar um dev Full-Stack. Estudar pra mim está sendo um prazer, passo horas codando e praticando o que aprendo e quando vejo já se passou mais um dia de aprendizado.

HTML é uma linguagem de marcação, onde toda a estrutura da página é criada, semânticamente.

CSS é a tecnologia que aplica a estilização a página criada em HTML.

A lógica e tratamento de dados é feito através do JavaScript.

SASS é um pré-processador de CSS. Com ele a folha de estilos é mais poderosa, podendo adicionar lógica, além de deixar o código mais organizado.

Vue.js é um framework progressivo leve e performático. Pode usá-lo para projetos de médio e longo porte.

Projetos:

🔗 Quotation Now

WebSite que consome API gratuita de cotação de moedas, a mesma atualiza a cada 30 segundos. Trouxe para a página como um 'catálogo de moedas', onde o cliente pode buscar os dados mais importantes. Tem uma página de catálogo, 'orçamento' de compra e venda, além do histórico, com máximo de 30 dias.

🔗 Pizza Place

WebSite de uma pizzaria, onde pode checar um cardápio e fazer pedidos (os mesmo são salvos em um banco de dados (json server)). Há também as funções de funcionário, onde poderá adicionar novos produtos, remover, além de atualizar. (por favor leia o README no github antes de prosseguir)

🔗 Petit Store

Uma loja de variedades, consumindo API para trazer os dados, é possível adicionar ao carrinho, remover do carrinho, além de filtrar os dados de acordo com categoria, a nota do produto e menor/maior preço.

🔗 Formulário de contato

Página de serviços para empresas, um formulário de contato, onde alguns campos são obrigatórios e o user somente avançará e poderá clicar o botão quando estes campos estiverem devidamente preenchidos.
